# TaiwanGiftFoodMenu  
這是一個Android的應用程式
## 功能：  
1.從政府資料開放平台抓取資料並整理排列  
2.搜尋資料  
3.點擊有興趣的資料可以顯示圖片  
4.尋找有興趣的伴手禮農產品
